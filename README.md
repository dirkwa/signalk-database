# signalk-database

Database / persistence plugin for [SignalK Node Server](https://signalk.org/).

**Status:** skeleton — name reserved on npm, implementation pending.

## Planned scope

Provide a single, configurable persistence layer for SignalK data:

- Store deltas and self-state in a time-series or relational backend
- Configurable retention and downsampling
- Query API for history and replay
- Backend-agnostic configuration (InfluxDB, QuestDB, SQLite, …)

## Install

Not ready for use. Once published with real functionality, install via the SignalK Appstore as `signalk-database`.
